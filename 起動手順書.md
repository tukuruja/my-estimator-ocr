# 統合見積システム（my-estimator-ocr）起動手順書

## このアプリでできること

| 機能 | 内容 |
|------|------|
| 見積タブ（二次製品工） | 施工延長・製品・機械・ダンプを選んで原価自動計算 |
| 見積タブ（擁壁工） | 擁壁高・延長から原価・人数を算出 |
| 見積タブ（舗装工） | 舗装幅・厚さから原価算出 |
| 見積タブ（撤去工事） | 撤去範囲から原価算出 |
| 図面OCR | PDF図面をアップロードして数量を自動抽出 |
| 見積レポート | 工種別見積書・単価根拠表の生成 |
| 単価表管理 | 機械・生コン・砕石・二次製品の単価一覧 |

---

## 動作環境

- **Node.js** 20 以上
- **npm** 9 以上
- **Python** 3.10 以上（OCR機能を使う場合）

---

## 初回セットアップ

```bash
# このフォルダに移動
cd my-estimator-ocr/

# 依存パッケージをインストール（初回のみ）
npm install --legacy-peer-deps
```

---

## 起動方法（開発モード）

```bash
# フロントエンド＋サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` を開くと使えます。

---

## OCR機能を使う場合（Python APIも起動）

```bash
# ターミナル1: フロントエンド
npm run dev

# ターミナル2: Python OCR API
cd ai_api/
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

---

## ビルド（本番環境）

```bash
npm run build
npm run start
```

## 画面構成（URLルーティング）

| URL | 画面 |
|-----|------|
| `/` | トップ（二次製品工 見積入力） |
| `/retaining-wall-input` | 擁壁工 入力 |
| `/pavement-input` | 舗装工 入力 |
| `/demolition-input` | 撤去工事 入力 |
| `/estimates/secondary-product` | 二次製品工 見積レポート |
| `/estimates/retaining-wall` | 擁壁工 見積レポート |
| `/estimates/pavement` | 舗装工 見積レポート |
| `/estimates/demolition` | 撤去工事 見積レポート |
| `/print` | 印刷用レポート |
| `/price-tables/machines` | 機械単価表 |
| `/price-tables/concrete` | 生コン単価表 |
| `/price-tables/secondary` | 二次製品単価表 |
