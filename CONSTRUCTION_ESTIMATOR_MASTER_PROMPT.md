### §2.1 外構入力 (exterior_work)
**定義**: 門柱・門扉・塀・花壇・アプローチ・駐車場等の外構全般
**主数量**: 面積(m²) or 延長(m) or 個数(箇所)
**計算フロー**:
```
掘削量 = 施工面積 × 掘削深さ × ほぐし係数L
砕石量 = 施工面積 × 砕石厚 × 1.2
コンクリート量 = 施工面積 × コンクリート厚 × 1.1
型枠面積 = 外周延長 × コンクリート厚 × 2
労務費 = (掘削日数 + 施工日数) × 作業員数 × 労務単価
材料費 = 砕石量×砕石単価 + コンクリート量×生コン単価 + 仕上材面積×仕上材単価
機械費 = 掘削日数 × 機械単価
```

### §2.2 型枠入力 (formwork)
**定義**: 各種コンクリート構造物の型枠工
**主数量**: 面積(m²)
**計算フロー**: 国交省土木積算基準準拠
```
型枠面積 = 入力面積(m²)
型枠材料費 = 型枠面積 × 型枠単価(円/m²)
型枠施工人工 = 型枠面積 ÷ 施工歩掛(m²/人日)
  歩掛基準: 基礎型枠=5.0m²/人日, 壁型枠=4.5m²/人日, スラブ型枠=6.0m²/人日
型枠労務費 = 型枠施工人工 × 労務単価
解体人工 = 型枠施工人工 × 0.4 (解体は組立の40%)
解体労務費 = 解体人工 × 労務単価
合計 = 型枠材料費 + 型枠労務費 + 解体労務費
```

### §2.3 土間コンクリ入力 (concrete_slab)
**定義**: 土間コンクリート・犬走り・基礎スラブ
**主数量**: 面積(m²)
**計算フロー**:
```
掘削量 = 面積 × (砕石厚+コンクリート厚+余掘0.1m) × ほぐし係数L
砕石量 = 面積 × 砕石厚 × 1.2
コンクリート量 = 面積 × コンクリート厚 × 1.1
ワイヤーメッシュ = 面積 × 1.1 (ラップ代10%)
目地材 = 目地延長(m) × 目地単価
型枠 = 外周延長 × コンクリート厚
伸縮目地 = 6m間隔で配置
労務費 = 打設人工 × 労務単価 + 左官人工 × 左官単価
```

### §2.4 フェンス入力 (fence)
**定義**: 各種フェンス(メッシュフェンス・目隠しフェンス・PCフェンス等)
**主数量**: 延長(m)
**計算フロー**:
```
支柱本数 = ceil(延長 ÷ 支柱間隔) + 1
  標準間隔: メッシュ=2.0m, 目隠し=2.0m, PC=2.0m
パネル枚数 = ceil(延長 ÷ パネル幅)
支柱基礎掘削 = 支柱本数 × (0.3m × 0.3m × 基礎深さ) × ほぐし係数L
基礎コンクリート = 支柱本数 × 基礎体積 × 1.1
支柱材料費 = 支柱本数 × 支柱単価
パネル材料費 = パネル枚数 × パネル単価
施工人工 = ceil(延長 ÷ 15) (15m/人日の歩掛)
労務費 = 施工人工 × 労務単価
```

### §2.5 ブロック設置入力 (block_installation)
**定義**: コンクリートブロック積み(CB工事)
**主数量**: 面積(m²) or 延長(m) × 高さ(m)
**計算フロー**: 国交省積算基準準拠
```
ブロック数 = ceil(面積 ÷ (0.39 × 0.19)) (標準CB 390×190mm)
基礎コンクリート = 延長 × 基礎幅 × 基礎厚 × 1.1
基礎型枠 = 延長 × 基礎厚 × 2
鉄筋量 = 縦筋(0.8m間隔) + 横筋(0.4m間隔) + 基礎筋
  縦筋: ceil(延長/0.8) × 高さ × D10単位重量(0.56kg/m)
  横筋: ceil(高さ/0.4) × 延長 × D10単位重量
モルタル充填 = ブロック数 × 0.003m³/個
目地モルタル = 面積 × 0.005m³/m²
施工人工 = ceil(面積 ÷ 4.5) (4.5m²/人日の歩掛)
労務費 = 施工人工 × 労務単価
```

### §2.6 型枠ブロック入力 (formwork_block)
**定義**: 型枠コンクリートブロック造(CP型枠ブロック)
**主数量**: 面積(m²)
**計算フロー**:
```
ブロック数 = ceil(面積 ÷ (0.39 × 0.19))
コンクリート充填量 = 面積 × ブロック厚 × 充填率0.65
鉄筋量 = 縦筋(0.4m間隔) + 横筋(0.2m間隔)
  D10: 0.56kg/m, D13: 0.995kg/m
基礎コンクリート = 延長 × 基礎幅 × 基礎厚 × 1.1
施工人工 = ceil(面積 ÷ 3.5) (3.5m²/人日 - CBより低い歩掛)
ブロック材料費 = ブロック数 × ブロック単価
コンクリート費 = 充填量 × 生コン単価
鉄筋費 = 鉄筋量(kg) × 鉄筋単価(円/kg)
労務費 = 施工人工 × 労務単価
```

### §2.7 自費工事入力 (self_funded_work)
**定義**: 任意項目の自費(実費)工事
**主数量**: 数量 × 単位(式/m/m²/m³/個/箇所)
**計算フロー**:
```
金額 = 数量 × 単価
(自由記述型 - ユーザーが数量・単位・単価を直接入力)
```

### §2.8 切盛土入力 (cut_fill)
**定義**: 切土・盛土工事(造成工事)
**主数量**: 体積(m³)
**計算フロー**: 国交省土木積算基準準拠
```
【切土】
切土量(地山) = 入力値(m³)
切土量(ほぐし) = 切土量 × ほぐし係数L
掘削日数 = ceil(切土量ほぐし ÷ 4時間当たり施工量)
残土搬出量 = 切土量ほぐし - 盛土利用量
残土搬出費 = ダンプサイクルタイム計算(AASHTO準拠)

【盛土】
盛土量(締固め後) = 入力値(m³)
必要土量(ほぐし) = 盛土量 ÷ 締固め係数C
転圧回数 = ceil(盛土量 ÷ (1層30cm × 面積))
転圧機械費 = 転圧日数 × 転圧機単価

【法面整形】
法面面積 = (法長 × 延長) ※法長=sqrt(高さ²+(高さ×法勾配)²)
法面保護工 = 法面面積 × 法面保護単価
```

## §3 既存計算ロジックのバグ・改善点

### BUG-1: 舗装工 worker capacity ハードコード
**場所**: calculations.ts 舗装計算
**問題**: `area / 120` が固定で材料種別・転圧機械に関わらず同じ
**修正**: 工種別歩掛テーブル参照に変更

### BUG-2: 擁壁工 ほぐし係数固定
**場所**: calculations.ts 擁壁計算
**問題**: groundConditionが未設定時に常に1.20固定
**修正**: getLooseningFactor()関数を統一使用

### BUG-3: 材料数量監査 density=0問題
**場所**: calculations.ts material_takeoff
**問題**: mode='t'でdensity=0のとき数量が0になる
**修正**: density未設定警告 + デフォルト値(2.3t/m³=コンクリート標準)

### BUG-4: 製品本数計算の端数処理
**場所**: calculations.ts secondary_product productCount
**問題**: decimal<=0.49で+2本、>0.49でceilだが根拠が不明瞭
**修正**: コメント追加（両端カット代で+2は業界慣行として正当）
